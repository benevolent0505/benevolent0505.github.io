Firefox OSハッカソンに参加してOpen Web Boardを貰ってきました
02-01-2015

昨日,1月31日にMozilla Japanのオフィスで行われたちょっぴり新春 Firefox OSハッカソン 2015に参加して来ました。例によって某後頭部の先輩からのお誘いがあったり、KDDIさんからOpen Web Boardを頂けるとの話があったりといった感じで軽いノリの参加でした。

前日まではOpen Web Boardにゲームパッドを指してそれを使ったゲームができたら面白いかなと考えていたのですが、どうも出来ないらしく断念。
当日はNHK番組表APIを使って指定した番組を予約できる様なアプリを作っていました(完成はしませんでしたが…)。

同じく参加していた同じ大学の先輩2人はサイコロの目にテレビ局を当てて、振った目の局で今やっている番組を表示するというものを作っていて、僕も番組を表示する部分を少しだけ手伝ったりしました。

ハッカソン的なイベントに参加するのは今回で2回目でしたが、とても楽しかったです。ただのYouTubeの動画をOpen Web Boardで流して音が出ただけでも歓声が湧いたり、サイコロを振ってちゃんと番組名が表示されただけで喜んだりして良い感じでした。
今回のようなゆるい感じのハッカソンだと気軽に参加できて楽しいので、こういうイベントの存在が広まって欲しいと思います。
